# n-value-calculator

CN인사이더 발주의 **1위안당 원화(N값)** 를 자동 계산하여 수입원가 구글시트의 지정 행에 입력하는 Claude 스킬.

## 무엇을 하나요

발주번호(`S-QEC...`)만 알려주면:

1. **CN인사이더 발주 상세** 에서 C(상품+물류+부가서비스), D(수수료)를 추출
2. **CN인사이더 CN출고** 에서 BL번호·발주 CBM 확인
3. **Gmail → 팝빌 청구서/정산서 PDF** 에서 환율(P), 해상운임(F), DOC·원산지·통관(G), 관세(I), 부가세(J) 추출
4. K = `C × 0.01 × P`, M = `(C+D) × P + SUM(E:L)`, **N = M / C** 자동 계산
5. 수입원가 구글시트의 다음 빈 행에 자동 입력 + 검증

## 주 트리거

- `1위안당금액 계산해줘`
- `위안당원화 계산해줘`
- `발주번호 [S-QEC...] N값 계산해줘`
- `[품목명] 1위안당 구해줘`

## 첫 사용 시 (세션당 1회)

다음 3가지를 한 번 받고 세션 동안 재사용:
1. 사업자 (이든코퍼레이션 / 이더컴퍼니 / 클린인테크 / 마인플로)
2. Gmail 계정
3. 팝빌 인증용 사업자번호

## 정상 N값 범위

| 품목 | 정상 N값 |
|---|---|
| 저단가(¥0.32, 수세미 등) | 320~360 |
| 중단가(¥5+, 캐리어 등) | 260~290 |
| 부피 큰 베개·거실화류 | 320~370 |

N < 250 또는 N > 400 이면 환율(P) 입력부터 재확인.
